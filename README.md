# BUS6440 Gold & Silver Safety Dashboard
This project is one of our course deliverables: a Python Dash dashboard that uses live daily market data from Yahoo Finance to analyze the behavior of the gold, silver, and equity markets.
Designed & Developed by: Junaid Hussain & Razkeen Irfan Shaikh

## Requirements
Install the dependencies from:
pip install -r requirements.txt

## Run
python main.py
